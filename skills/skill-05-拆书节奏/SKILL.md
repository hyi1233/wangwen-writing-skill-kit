# 网文拆书与节奏

> 核心信条：找可复制的，不仿神作。仿技法不仿剧情。

## 触发条件

- 拆解某本书 / 某类文
- 学习网文写作技法
- 做写作规划 / 设计章节奏
- 制定仿写计

## 核心则

- **找可复制的，不神作** — 三江榜、L5写手的书可复制。《诡秘之主》《剑来》等神作独一份
- **扫榜看共点** — "有一个人写成，有第二个，有十个，有一百个" —— 共同模式才值得学
- **细粮 vs 粗粮** — 粗粮（三江/中腰部）学签约模板，快速入门；细粮（L5/头部）提升写作上限
- **仿技法不仿剧情** — 学"怎么写的"，不是"写了什么"

## 五步拆工作流

### 第一步：选书
1. 找三江榜、L5写手的书
2. **通过金手指类型选书，不是通过题材选**
3. 不仿不可复制的神作
4. 扫看共同点

### 第二步：通读（约10章）
记录：事件团 / 阶段性目标与开篇关联 / 背景设定 / 金手指体系设定 / 人物设定与人设弧光

### 第三步：重点信息提取
关注：数值化对比、背景信息引导、详略技巧、行为修饰形容词（倾向性）、契诃夫之枪、主角反馈总结

### 第四步：写设定集（不可跳过！）
三要素，缺一不可：
1. **人设** — 主角行为模式（怎么说话、怎么决策） + 配角脸谱化
2. **事件核心** — 明确后续要达成的结果（落点已知才能倒果为因）
3. **金手指功能逻辑** — 正反两面分析：能干、不能干、代价

> 不做设定集 = "梦到哪句写哪句" = 剧情散

### 第五步：仿写（倒果为因）
1. 先理清事的最终落点
2. 在前文制造起伏和落差
3. 所有跌宕都基于此逻辑展开
4. 技法拿走，用在完全不同的故事里

## 节奏控制技法

### 循环拆解（收获→期待→演绎）
每1-2章完成一个循环，形成自动驱动的节奏引擎：

| 步骤 | 内容 |
|------|------|
| **收获** | 主角获得某样东西（经验/资源/情报）→ 读者满足感 |
| **期待** | 收获同时引出新目标/危机 → 读者产生新期待 |
| **演绎** | 主角为新目标行动 → 读者跟随剧情 |

循环完成后立即开始下一轮，不要停顿。

### 黄金三结尾两种
- **方式A**：开启新危机（三章完成开篇设置，最后出现更大危机）
- **方式B**：展示金手指（三章铺垫完，金手指第一次发挥作用）

### 断方式
- **卡期待**：给到变强期待/解决方法，卡在希望点
- **卡转折**：金手指解决了困境，出现新引线下章继续
- **危机断**：断在"刀到脖子"或"颠覆认知"的瞬间。下一开头必须快速释放

### 上强度技巧
1. 先写主角生理/心理反应（汗毛倒竖、想跑、胃里一紧）
2. 再写事物外形/威胁细节
3. **先反应后对象** = 读者先感受恐惧，再看到恐惧源

### 平静与爆发交替
- 爆发后必须有平静（让读者消化信息）
- 平静后必须有爆发（否则无聊）
- 平静章节可以推进设定/伏笔/人物关系

### 章情绪标注式细纲
章/每个情节点标注情绪标签 → 情绪标签连成曲线 → 检查有没有连续N章同情绪

## 特殊题材拆解

### 同人写作四要素
1. **节点选择** — 选原作的哪个时间/事件点切入
2. **主角选择** — 穿越成谁？原创角色还是原著人物？
3. **风格还原** — 模仿原作的叙事风格和语言习惯
4. **投其所好** — 写原著读者最想看的场景/展开

### "螺丝壳里做道场"
循环流/重生流：选定有限要素（固定场景/角色群/事件类型），在要素内做无限演绎。每次循环递进（不是重复），比上次更深/更难/更有趣。

### 认知税减免
使用读者已知的文化资产（名著/名人/历史事件）来减免理解成本。高考/职场/春运都是普适性参照。

## 快速自检

- [ ] 设定集完成了吗？（人设+事件核心+金手指逻辑）
- [ ] 通过金手指类型选书，不是题材？
- [ ] 倒果为因做了吗？（先定落点→前文制造起伏）
- [ ] 仿的是技法不是剧情？
- [ ] 循环节奏设计好了吗？（收获→期待→演绎）
- [ ] 断章点在关键节点吗？
- [ ] 情绪曲线有波动吗？
- [ ] 选的书可复制吗？
