## 十三、实战自检清单

### 13.1 写作前
- [ ] 设定集完成？人设+事件核心+金手指逻辑三要素齐全？
- [ ] 简介三步完成？背景事件→金手指→目标走向？
- [ ] 黄金三章简纲倒果为因推导完成？
- [ ] 六要素（舞台/困境/解法/信息同步/高光/牵引）规划好了？

### 13.2 开篇自检
- [ ] 开篇几百字信息密度够高？
- [ ] 驱动在几百字之内出现了？
- [ ] 原主事件延展性建立了？（睁眼穿）
- [ ] 金手指在前三章有应用场景？
- [ ] 前文铺垫的信息在后文都有引爆？
- [ ] 没有与主线无关的废话？

### 13.3 行文自检
- [ ] 每段剧情后有没有主角主观反馈？
- [ ] 穿插设定后有没有回归正在进行时？
- [ ] 用到什么写什么？用不到的不写？
- [ ] 对话是不是废话？删掉后影不影响主线？
- [ ] 核心事件没尘埃落定前，没有开启下一个事件？
- [ ] 下行剧情是不是为主角上行爽点做铺垫？

### 13.4 去AI化自检
- [ ] 有没有解释尾巴？
- [ ] 情感有没有附着物？
- [ ] 人物有没有缺点/味道？
- [ ] 目的是不是太露？
- [ ] 有没有AI高频词？
- [ ] 比喻喻体是不是生活真实存在？
