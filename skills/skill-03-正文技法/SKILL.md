# 网文正文技法

> 覆盖六大技法层，69 张知识卡片。写作时按需组合。

## 触发条件

- 写正文 / 续写章节 / 修改文段
- 设计爽点 / 编排打脸剧情
- 塑造角色 / 设计出场
- 埋设伏笔 / 设计反转

## 六大技法总览

### 叙事层（15 技法）

| 技法 | 核心要点 |
|------|----------|
| **倒果为因** | 先确定爽点落点，逆向推导铺垫。网文是倒着写出来的 |
| **信息差** | ①读者+主角知道/配角不知（最常用，信息差=窃喜感）；②读者不知（惊讶感）；③读者不知/配角不知（反转） |
| **普适性类比** | 用群体熟知的参照物做类比（高考/春运/职场），降低理解成本 |
| **行为跟进逻辑** | 前句行为，后句行为逻辑。"不会游泳"→"他保持双手紧贴裤腿漂浮，一动不敢动" |
| **逻辑堵路** | 逐条堵死退路让主角走向预设路线。看起来是主角选的，实际是逻辑必然 |
| **信息锚点** | 具体可感元素替代抽象形容词。一段最多2-3个锚点 |
| **倾向性描写** | 给出预测方向但不明说，让读者自行推断。"故作吃惊"≠"满脸吃惊" |
| **锚点写作法** | 循环/重生流中用有限要素（小杰+风）做状态标记。锚点可变升级 |
| **物理转场** | 空间位移（走/跑/飞）完成转场，保持时空连续性。禁用"与此同时" |
| **多重视角** | 同一事件切换视角，每视角有独立信息价值。主视角80%以上篇幅 |
| **信息留白** | 前文充分建立前提后，后续省略说明，读者自动补全 |
| **一笔三用** | 一个细节 = 推进剧情+塑造人设+埋伏笔。网文不允许零功能文字 |
| **没工夫看** | 完成任务但不看结果，后续补充产生反差爽感 |
| **路人发射器** | 每个路人闲聊都是功能性的——背景/预警/情绪对照/伏笔 |
| **复合结构** | 演绎层70%（含蓄，给老书虫）+ 解析层30%（明确，给小书虫） |

### 情绪层（10 技法）

| 技法 | 核心要点 |
|------|----------|
| **困境+希望** | 成对出现。无希望的困境=吃屎；无困境的希望=意淫 |
| **爽点链条** | 困境→有手段(信息差)→压迫→压力累积→行动→反转打脸→收获 |
| **反向 Flag** | Flag由配角说出，与主角已知信息完全相反。Flag越绝对反转越爽 |
| **情绪转折** | 金手指在别人否定后觉醒，比前觉醒多一层转折情绪 |
| **递进下压** | 五步阶梯：①进门冷遇→②偏心对待→③求援被拒→④梦想打压→⑤情感核弹 |
| **时间错位** | 先展示悲剧结局，再回到平静。必须给"这次也许能改变"的希望 |
| **意难平** | 找到最让读者"不该如此"的不甘点，全书围绕此展开 |
| **拉扯感** | 爽点即将爆发时插入次要事件延迟满足（半章-一章） |
| **明抑暗扬** | 明线打压，暗线埋伏笔/优势。抑不是真输，是拉高后续弹跳系数 |
| **对照组** | 起点相同但走了不同路，命运对比展示变化 |

### 爽感层（10 技法）

| 技法 | 核心要点 |
|------|----------|
| **网文相对论** | 爽感=对比。参照物越强，踩上去越爽 |
| **维度碾压** | 我越弱我越强：不降维打击，是升维打击（修仙打武道=维度不同） |
| **欲扬先抑** | 下行必须真实，上行必须有根据。差距=爽感系数 |
| **关我何事** | 困境与主角切身利益绑定。自保入局＞发善心入局 |
| **事件强度** | 不弱化对手，用更强敌人衬托。配角打三星怪勉强，主角打五星+六星 |
| **反差-反转** | 建立强烈预期→完全相反打破。反转后让读者觉得"前文暗示过了" |
| **插旗必反转** | 配角立的Flag，读者知道会打破但期待"怎么打破" |
| **弹幕最多的打法** | 选读者最想看的剧情方向，倒果为因做前置铺垫 |
| **七字诀** | 删"是"字加动作；伏笔像葡萄串；情绪直捅人性窝 |

### 人物层（12 技法）

| 技法 | 核心要点 |
|------|----------|
| **主观视角四步** | 感知→反馈→动机→驱动。90%新人丢"反馈"这步，这是网文重中之重 |
| **演绎不靠定义** | 不写"主角是个狠人"→通过表情动作行为演绎 |
| **脸谱化要快** | 非长线角色一两句话完成，减少节奏断档 |
| **参照物打脸** | 没有参照物的高光打脸不可能成立。具体到人，具体到事 |
| **标签组合** | 外貌标签(矛盾特征)+行为标签(高频动作)+台词标签(口头禅) |
| **差异化反应** | 一场戏收尾不能所有人反应一致。简单者被说服，最聪明的一言不发 |
| **异常正常化** | 给主角异于常人的特质（如情感感知障碍），承载超验体验 |
| **先解构再重构** | 先展示世俗/庸俗面，关键时刻展现神秘力量。反差即力量 |
| **语言DNA** | 泼辣碎嘴型/沉默定音型/精明市侩型/少年克制型，遮名也能辨 |
| **女主不白给** | 给足逼格和好人设，通过男主强大不断吸引 |
| **人物出场** | 标签动作+关系定位+一句闲笔 = 28字完成人物锚定 |
| **女描写锚点** | 不写"很美"，写大长腿/脚趾涂红指甲等具体锚点 |

### 伏笔层（10 技法）

| 技法 | 核心要点 |
|------|----------|
| **三层次** | 短线（3章内回收）/中线（阶段目标）/长线（贯穿全文） |
| **反向伏笔** | 给角色设无能标签，读者以为某类人→情绪高点引爆揭露 |
| **契诃夫之枪** | 只要写了就必须有用。看描写篇幅而非事物大小 |
| **草蛇灰线** | 不影响阅读体验的情况下铺线 |
| **一事不烦二主** | 角色/设定出现=后续必有明确用途 |
| **信息闭环** | 标签引用、背景铺垫，前后形成闭环 |
| **结论锚点** | 前文通过事件建立某角色/物品权威性，后文一句话就能让读者接受新结论 |
| **葡萄串** | 一个细节牵连多个后续，像葡萄一串 |
| **主线与短线合并** | 短线事件+主线伏笔在情绪高点合并产生高光 |
| **死人即伏笔** | 从尸体上搜出关键物品推动剧情 |

### 文字层（12 技法）

| 技法 | 核心要点 |
|------|----------|
| **删"是"字加动作** | "李云是山贼"→"刀光一晃，骂声从身后炸来" |
| **动词精准** | 精准小动词＞三个形容词 |
| **生活化比喻** | "就像高考"＞"犹如命运齿轮转动" |
| **多感官细节** | 视觉/听觉/嗅觉结合沉浸感 |
| **三层语言** | 叙述层(普通话)+对话层(方言词汇)+内心独白(因身份切换) |
| **短-短-长-短** | 交替节奏，连续三句以上长句即违规 |
| **手术刀分段** | 一段只讲一个动作，手机屏幕不超过3行 |
| **拟声词独立行** | 充当节奏鼓点，不可埋在句中 |
| **恐怖物理递进** | 听觉→震动→视觉，最后给视觉且只给局部 |
| **恐怖需有触觉** | 湿/冷/粘 |
| **方言调味** | 每段植入2-3个标志性词汇，过多则阅读障碍 |
| **短句定格** | 恐怖/反转最后一句必须短句定格，像快门"咔嚓" |

## 六阶段写作工作流

### 阶段 1：剧情骨架设计
- 倒果为因：先定最终爽点落点
- 信息差：读者+主角知道什么？配角不知道什么？
- 逻辑堵路：把不该走的路线封死
- 确认：一事不烦二主，写的每样东西后文有应

### 阶段 2：人物交互设计
- 主观视角四步：感知→反馈→动机→驱动
- 配角脸谱化：一两句话完成（外貌标签+行为标签）
- 主角演绎：每章字里行间体现人设
- 差异化应：一场戏不同角色不同反应

### 阶段 3：情绪节奏铺设
- 困境+希望成对出现
- 递进式下压（五步阶梯逐步加码）
- 信息差：让读者知道主角有翻盘手段
- 明抑暗扬：正面打压+暗线拔高

### 阶段 4：爽感爆发
- 参照物/对照组设计
- 反差反转：建立预期→打破预期
- 事件强度拉升：配角打三星怪勉强，主角打五星+六星
- 围观视角衬托：通过配角反应衬托主角高光

### 阶段 5：伏笔引线铺设
- 三层次伏笔（短/中/长）
- 结论锚：前文建立权威，后文一句话接受新结论
- 葡萄串：一个细节牵连多个后续
- 事件收尾给下一章引线（不要"回房睡觉"）

### 阶段 6：文字打磨
- 删"是"字加动作
- 短-短-长-短节奏交替
- 多感官细节（不只写看到的）
- 检查信息锚点（一段≤3个锚点）
- 检查物理转场（禁用"与此同时"）

## 快速自检

- [ ] 每个细节是否有≥1功能？（纯装饰文字→删）
- [ ] 困境+希望是否成对？
- [ ] 主观反馈四步是否完整？
- [ ] 参照物/打脸对象是否具体到人到事？
- [ ] 信息差是否制造了期待？
- [ ] 有无反向Flag？
- [ ] 伏笔是否有后续呼应？
- [ ] 角色是否有差异化反应？
- [ ] 有无"是"字？有无空洞形容词？
- [ ] 转场是否用物理位移？
- [ ] 事件收尾是否给了下一章引线？
