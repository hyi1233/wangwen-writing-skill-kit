# 去AI化写作

> 核心认知：人类写完动作立刻停笔。AI写完必跟解释。

## 触发条件

- 去除AI味 / 检查文字质量 / 文笔润色
- 诊断AI特征 / 让文章更自然

## 核心认知

### 叙事自信 vs 解释强迫症

| 人类高手 | AI 写作 |
|---------|---------|
| 相信"事情"本身就有力量 | 生怕读者看不懂，紧跟解释 |
| 只摆画面、动作、对话，停笔 | 写完必跟"这表现了……""仿佛……" |
| 留白让读者自己领会 | 事无巨细全说透 |

**检验**：删掉解释句，读者还能get到原意吗？能 → 解释多余。

### AI 四大毛病

1. **滥用形容词** — 璀璨、熠熠生辉 → 用最简单的普世语言
2. **过度解释** — 动作后必跟"仿佛""这表现了" → 写完立刻停笔
3. **情感悬浮** — 空洞的决心 → 把情感附着在具体物体上
4. **完美角色** — 无缺点无异味 → 加点缺点、异味、尴尬

## 十步诊断工作流

### Step 1：扫描AI特征

检查8项：解释尾巴、形容词堆叠、类比比喻、情感悬浮、完美角色、上帝表达、抽象概念、空手角色。

### Step 2：斩断解释欲

- 删"这表现了……""可以看出……"
- 删"仿佛/宛如/好似/犹如"引导的类比
- 写完动作 → 句号 → 切下一段
- 读者没看懂 → 回去改铺垫，不解释

⚠️ 不误删"行为跟进逻辑"（C-4），那是展示逻辑，不是旁白解释。

### Step 3：情感物化

| 抽象情感 | 具体附着物 | 示例 |
|---------|-----------|------|
| 思念 | 饺子 | "端起那碗凉透的饺子，吃了两口，吃不下了" |
| 遗憾 | 旧日历 | "墙上的日历还停在三月五号。从那天起就没再翻过" |
| 爱 | 鱼腥味 | "一股鱼腥味从他袖口钻进鼻孔" |
| 初恋 | 背影 | "那个消失在人海中的身影" |

**核心**：情感必须绑定一个具体物/气味/声音。

### Step 4：生理反应替代

| 情感 | 生理反应 |
|------|---------|
| 愤怒 | 咬肌发紧、指甲抠进掌心 |
| 恐惧 | 胃部抽搐、后颈发凉、汗毛倒竖 |
| 悲伤 | 喉咙发紧、眼眶发酸但不流下来 |
| 恶心 | 胃酸上涌、干咽、嘴角不受控地抖动 |

⚠️ 生理反应和心理描写不要同时出现，重复了。只保留生理反应。

### Step 5：角色添加真缺点

- 不写"太善良"这种假缺点
- 异味：汗臭/口臭/鱼腥味
- 行为尴尬：撒谎摸耳垂/打呼噜像拉锯
- 功能性缺点：与剧情相关的真毛病

### Step 6：零"像"字铁律

- "像一头猛虎" → 写行为（咆哮、猛扑、双眼充血）
- "美得仿佛仙女" → 写锚点（白裙、赤足、脚踝上的红绳）
- "像"字是AI味第一识别标记

### Step 7：杜绝上帝表达

- 不写"他是个好人" → 让读者通过行为自己判断
- 不写"注定改变一切" → 写事实，让读者自己感受
- 对话里不加作者旁白评价

### Step 8：具体化

| 抽象 | 具体 |
|------|------|
| "他很穷" | "碗里的粥稀得能照出人影" |
| "天气炎热" | "石板路的鸡蛋自己就能煎熟" |
| "战场惨烈" | "断臂的士兵在泥水里爬，肠子拖了三尺远。没人扶他" |

### Step 9：潜台词对话

- "你最近很忙。" 表面观察，潜台词：你背着我做了什么
- 他不答。低头把茶碗转了半圈。 潜台词：是的，被你说中了
- 用动作/停顿/语气暗示真正意思

### Step 10：道具绑定

角色出场不空手：拿杯子/擦刀/剥橘子/盘珠子/系扣子
手里动作反映心理：紧张→捏紧，放松→转动

## 禁用词清单

扫描以下词汇，发现后必须用动作/物体/反应替代：

| 类型 | 禁用词 |
|------|--------|
| 形容词 | `璀璨` `熠熠生辉` `深邃` `凛冽` `凛然` `一丝` `一抹` |
| 比喻词 | `像一样` `仿佛` `宛如` `好似` `犹如` `宛若` |
| 情绪旁白 | `心中涌起` `内心感到` `内心深处` `眼神中透露` `嘴角勾起一抹冷笑` `五味杂陈` |
| AI高频词 | `然而` `不禁` `顿时` `瞬间` `暗暗` `不由自主` `鬼使神差` |
| 套路句式 | "眼中闪过一丝""心中暗暗发誓""一股…涌上心头""脸上露出了…的表情" |

## 14条快速自检

| # | 检查项 | 通过标准 |
|---|--------|---------|
| 1 | 解释尾巴？ | "这表现了""仿佛""内心感到" → 删 |
| 2 | 情感有附着物？ | 写"伤心"后跟了具体物象 |
| 3 | 人物有味道？ | 有异味/缺点/尴尬 |
| 4 | 目的太露？ | 不是喊口号，是卑微动作 |
| 5 | 有"像/仿佛/宛如"？ | 全删，换成动作/物体/反应 |
| 6 | 滥用形容词？ | 换成最简单语言 |
| 7 | 上帝表达？ | 让读者自己判断 |
| 8 | 抽象概念？ | "很穷""很热" → 3个具体细节 |
| 9 | 用生理反应表达？ | 恐惧→胃抽搐/愤怒→咬肌发紧 |
| 10 | 角色手里有东西？ | 不空手 |
| 11 | 对话有潜台词？ | 角色不直接说出全部 |
| 12 | 写完动作立刻停笔？ | 句号后切下一段 |
| 13 | 生反应vs心理描写重复？ | 只保留生理反应 |
| 14 | 是真缺点？ | 不是"太善良"这种假缺点 |
