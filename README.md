# 网文写作技能包 (wangwen-writing-skill-kit)

> 基于B站 UP 主「网文徐掌柜」23 本拆书、23 万字笔记蒸馏的完整网文创作方法论系统
>
> **统一入口、按阶段加载、不爆上下文**

---

## 它能做什么

一个总调度 Skill `webnovel-creator`，自动识别你的创作阶段，按需加载对应的方法论。覆盖网文创作全流程 14 个模块。

你说需求 → Skill 判断阶段 → 只加载需要的文件 → 方法论指导

| 你说… | 加载的模块 | 覆盖内容 |
|------|-----------|---------|
| "网文怎么写" / "写作原则" | 核心理念 | 10 条创作哲学 |
| "拆书" / "仿写" | 拆书方法论 | 六步法、记录清单、仿写流程 |
| "开篇" / "简介" / "金手指" | 开篇架构 + 金手指设计 + 简介写作 | 三种开篇、黄金三章、五大金手指类型、简介模板 |
| "叙事" / "伏笔" / "情绪" / "人物" | 叙事技巧 + 伏笔铺垫 + 情绪爽感 + 人物塑造 | 倒果为因、契诃夫之枪、拉弓理论、脸谱化 |
| "去AI化" / "AI味" | 去AI化 | 斩断解释欲、情感物化、10 项自检清单 |
| "自检" / "金句" | 实战自检清单 + 金句集 | 4 类检查项、50+ 条原话金句 |

## 安装到 Claude Code

### 方式一：直接作为 Skill 安装

```bash
git clone https://github.com/hyi1233/wangwen-writing-skill-kit.git
cp -r wangwen-writing-skill-kit/skills/webnovel-creator ~/.claude/skills/
```

### 方式二：作为插件安装

```bash
git clone https://github.com/hyi1233/wangwen-writing-skill-kit.git
cp -r wangwen-writing-skill-kit ~/.claude/plugins/
```

重启 Claude Code，说一句"帮我写个网文开篇"验证是否生效。

### 设计原理：渐进式披露

传统做法是把所有内容塞一个文件（673行），加载时全部进入上下文，浪费 token。

现在的设计：
- **SKILL.md** 只做路由（70行），始终加载
- **references/** 按需加载，每次只读 1-3 个文件（100-200行）
- 上下文始终控制在 200-300 行以内
