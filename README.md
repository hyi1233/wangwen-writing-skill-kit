## 📦 安装到 Claude Code

```bash
git clone https://github.com/hyi1233/wangwen-writing-skill-kit.git
cp -r wangwen-writing-skill-kit/xzhang-plugin ~/.claude/plugins/
```

重启 Claude Code，技能自动生效。无需额外配置。

**安装验证**：在 Claude Code 中说一句"帮我写个网文开篇"，如果能触发完整的开篇设计工作流，说明安装成功。

## 🚀 快速上手

### 1. 安装（10 秒）
克隆仓库 → 复制到插件目录 → 重启 Claude

### 2. 使用
直接用中文说需求，例如：
- 帮我设计一个错位金手指，要求有代偿机制"
- "这段文字有 AI 味，帮我改一下"
- "帮我拆解《XXX》这本书的写作技巧"
- "写个黄金三章的开篇，主角穿越成一个废柴"

### 3. 查细节
想深入某个技法时，去 `knowledge-cards/` 文件夹找到对应的卡片文件，里面有完整的定义、规则、案例和常见错误。
