### 卡片 F-1：主观视角黄金流程

| 字段 | 内容 |
|------|------|
| **关键词** | 主观视角、感知反馈、四步流程 |
| **定义** | 主角视角的标准处理流程：感知 → 反馈 → 动机 → 驱动。这是网文90%篇幅必须遵循的黄金流程。 |
| **规则** | 1. **感知**：主角通过感官接收外界信息（看到/听到/闻到）<br>2. **反馈**：内心加工、判断、产生情绪（"这不正常"）<br>3. **动机**：情绪/判断升级，形成行动动机（"必须查清楚"）<br>4. **驱动**：动机转化为具体行动（推门进去）<br>5. "主角内心反馈是写一本网文的重中之重" |
| **案例** | 感知："他闻到空气中有一股甜味，但很淡。" → 反馈："不对，这是毒。但剂量不够杀人。" → 动机："有人想毒害但不想闹大——内部人干的。" → 驱动："他不动声色，开始排查今天来过的人。" |
| **常见错误** | 1. 跳过了反馈直接驱动——主角像程序一样执行"看到A就做B"，没有思考过程<br>2. 感知了但没有反馈和动机——主角变成摄像头 |
| **关联卡片** | A-9, C-4, B-11 |

### 卡片 F-2：人设靠演绎不靠定义

| 字段 | 内容 |
|------|------|
| **关键词** | 人设演绎、行为刻画、不靠定义 |
| **定义** | 人设不是作者定义的（"他是个狠人"），而是通过角色行为、语言、表情、背景从各角度演绎出来的。 |
| **规则** | 1. 不靠作者强行定义，全书用章节刻画人设<br>2. 从行为、语言、表情、动作、背景多角度体现<br>3. "谜底就在谜面上"——读者从角色做的事自己得出结论<br>4. 定义式人设是AI写作最大的标志之一 |
| **案例** | 定义式："他是个冷面无情的人。"（读者不信）<br>演绎式："他把铜钱丢在桌上，看都没看对方。'滚。'语气淡得像说今天的天气。"（读者自己得出"冷面无情"的结论——而且信了） |
| **常见错误** | 1. 用作者旁白宣布人设——"他是个好人"——读者无感<br>2. 角色行为和"定义"矛盾——定义说冷酷，行为却优柔寡断 |
| **关联卡片** | F-6, H-10, A-7 |

### 卡片 F-3：脸谱化要快

| 字段 | 内容 |
|------|------|
| **关键词** | 脸谱化、快速刻画、配角效率 |
| **定义** | 配角的人设脸谱化必须快——一两句话完成，把笔墨留给主角。 |
| **规则** | 1. 配角出场一两句话完成标签化（外貌+态度+功能）<br>2. 只有后续有重头戏的配角才值得细化<br>3. 主角必须细致，体现到每章字里行间<br>4. 配角不能是废话，出场时必须同时给出信息 |
| **案例** | "赵三，五十来岁，一脸横肉，是城东出了名的泼皮。"——一句话完成脸谱化。外貌（一脸横肉）+身份（泼皮）+定位（城东）。后续如果他有重头戏再细化。 |
| **常见错误** | 1. 配角写了三页外貌描写，但后续戏份两句带过——本末倒置<br>2. 配角脸谱化太模糊——读者分不出资深配角A和配角B |
| **关联卡片** | F-2, F-10, C-14 |

### 卡片 F-4：参照物/对照组

| 字段 | 内容 |
|------|------|
| **关键词** | 参照物、对照组、对比爽感 |
| **定义** | 设立明确的参照物角色，用来反衬主角的处境和未来的逆袭。没有参照物就没有打脸爽感。 |
| **规则** | 1. 参照物必须"具体到人，具体到事"——不要泛泛的环境压抑<br>2. 参照物拥有主角想要却得不到的资源<br>3. 参照物对主角不友好，施加压迫<br>4. 最终参照物被主角踩在脚下——这就是爽点 |
| **案例** | 参照物：同族天才少年，灵根上成，爷爷宠爱，资源倾斜。主角：灵根下成，爷爷冷脸，例银减半。参照物每次出现都在提醒读者"主角被亏待了多少"，为后续反转蓄力。 |
| **常见错误** | 1. 没有具体参照物，只有泛泛的"世道不公"——没有具象仇恨<br>2. 参照物太坏/太蠢——读者觉得主角赢他是应该的，不觉得爽 |
| **关联卡片** | E-1, D-5, D-10 |

### 卡片 F-5：标签组合法

| 字段 | 内容 |
|------|------|
| **关键词** | 标签组合、人设构建、立体化 |
| **定义** | 通过外貌、行为、台词三层标签叠加，构建立体可识别的角色人设。 |
| **规则** | 1. **外貌标签**：矛盾或突出特征（白面书生但满手老茧）<br>2. **行为标签**：绑定高频动作（每次紧张就摸鼻子）<br>3. **台词标签**：标志性口头禅或说话风格<br>4. 三层叠加后角色自带辨识度，读者隔了三章还能认出来 |
| **案例** | 外貌：瘦长脸，左脸一道疤<br>行为：说话前习惯性眯眼睛<br>台词："这事儿……得从长计议。"（每次遇到危险都这么说）<br>三层叠加 = 一个谨慎、有经验的老江湖形象。 |
| **常见错误** | 1. 只有外貌标签，行为台词完全没有个人特征<br>2. 标签太多（外貌8个+行为5个+台词3个）——记不住 |
| **关联卡片** | F-3, F-7, F-12 |

### 卡片 F-6：差异化反应

| 字段 | 内容 |
|------|------|
| **关键词** | 差异化反应、群像、不同性格不同反应 |
| **定义** | 一场戏收尾时，让不同角色基于自身性格给出不同反应，避免所有人一个态度。 |
| **规则** | 1. 不能让所有人都有一种反应——"所有人都惊呆了"是偷懒<br>2. 头脑简单的→被说服/被吓到<br>3. 最聪明的→一言不发，"冷冷地看着"（留悬念）<br>4. 中间层的→有自己的小算盘 |
| **案例** | 主角一拳打飞了对手。<br>头脑简单的围观者："哇，太厉害了！"（被折服）<br>老谋深算的长老："……"（冷眼评估主角威胁等级）<br>和主角有旧怨的："哼，运气好。"（不愿承认）<br>——同一个事件，四种反应，每个反应都暗示后续剧情走向。 |
| **常见错误** | 1. 所有人同一反应——"全场寂静"——单调且浪费机会<br>2. 差异化反应不符合角色人设——胆小的居然第一个鼓掌 |
| **关联卡片** | F-4, E-1, C-14 |

### 卡片 F-7：异常正常化

| 字段 | 内容 |
|------|------|
| **关键词** | 异常正常化、异于常人、特质设定 |
| **定义** | 给主角一个异于常人的特质，但对主角来说这个特质是"正常"的——这种异常正常化让角色自带亮点。 |
| **规则** | 1. 主角拥有某个异常特质（感知力超强/不怕黑/味觉异常）<br>2. 主角对此习以为常，不当回事<br>3. 配角看到后会震惊，制造信息差和反差<br>4. 异常特质最好在后续剧情中有实际功能（不只是人设装饰） |
| **案例** | 主角能听到别人心跳声——他自己觉得"大家都听不到吗？"（异常正常化）。后来在赌石/谈判中，这个能力帮助他判断对方是否撒谎。<br>主角不怕痛——穿越前就有先天性无痛症。穿越到异世界后，别人受伤要停，他继续打。 |
| **常见错误** | 1. 异常特质没有任何剧情用途——纯装饰<br>2. 主角对自己的异常太过在意——不像"正常化" |
| **关联卡片** | B-7, F-5, E-10 |

### 卡片 F-8：先解构再重构

| 字段 | 内容 |
|------|------|
| **关键词** | 解构重构、先世俗后神秘、认知阶梯 |
| **定义** | 写神秘/超凡概念时，先用世俗的语言/场景解构它，让读者理解底层，再重构出神秘感。 |
| **规则** | 1. 神秘概念 → 用读者懂的事物类比（解构）<br>2. 理解底层逻辑后 → 再用神秘/超凡语言包装（重构）<br>3. 先让读者"懂了"，再让他"不明觉厉"<br>4. 直接写神秘 = 读者看不懂；只写世俗 = 读者觉得无趣 |
| **案例** | 修仙的"灵气"：先解构成"就像空气，但只有修炼过的人能吸到。"→ 读者理解后 → 再重构："灵气如水波荡漾，汇聚于丹田。"<br>金手指的"经验值" = "就像打游戏杀怪升级"（解构）→ "功法运转一周天，体内真元凝实一分"（重构）。 |
| **常见错误** | 1. 全是解构没有重构——"就是打游戏升级嘛"——失去神秘氛围<br>2. 没有解构直接重构——一堆术语，读者一头雾水 |
| **关联卡片** | B-10, C-3, I-3 |

### 卡片 F-9：四种语言DNA模板

| 字段 | 内容 |
|------|------|
| **关键词** | 语言DNA、说话风格、角色区分 |
| **定义** | 不同角色应有独特的说话模板（语言DNA），读者不看台词标识就知道谁在说话。 |
| **规则** | 1. **模板A（粗犷型）**：短词、命令式、带俚语——"少废话，动手。"<br>2. **模板B（书卷型）**：长句、引经据典、迂回——"此事恐有蹊跷，依老夫之见……"<br>3. **模板C（市井型）**：接地气、比喻俗、节奏快——"跟你说白了吧，就是钱的事儿。"<br>4. **模板D（阴冷型）**：话少、停顿多、语气平——"……你会后悔的……" |
| **案例** | 同一个意思"别去"：<br>粗犷型："站住！找死？"<br>书卷型："阁下三思，此去凶多吉少。"<br>市井型："兄弟你脑子进水了？那地方能去？"<br>阴冷型："……你确定……去了还能回来？" |
| **常见错误** | 1. 所有角色说话风格一样，只看标签才知道谁在说<br>2. 角色说话风格和人设矛盾（粗人是文学腔调） |
| **关联卡片** | F-5, I-5, H-13 |

### 卡片 F-10：女主人设不白给不倒贴

| 字段 | 内容 |
|------|------|
| **关键词** | 女主、白给不倒贴、独立人设 |
| **定义** | 女性角色的核心原则：不能白给（无缘无故对主角好），不能倒贴（抛弃一切追主角）。 |
| **规则** | 1. 女主对主角的好必须有合理原因（利益交换/感恩/共同目标）<br>2. 女主必须有自己的人生目标和行动线<br>3. 感情线建立在共同经历/互相认可上，不是"看到主角第一眼就爱上"<br>4. 女主有拒绝主角的能力和意愿——她的选择才有意义 |
| **案例** | 好的写法：女主是商会大小姐，和主角合作做生意。合作中认可主角的能力和品格，感情自然产生。<br>差的写法：女主看到主角打了一场，"此生非你不嫁"——白给+倒贴。 |
| **常见错误** | 1. 女主出场就是工具人/挂件，没有独立目标<br>2. 多个女性角色同时倒贴主角——后宫文的毒点 |
| **关联卡片** | F-4, F-2, D-7 |

### 卡片 F-11：人物出场公式

| 字段 | 内容 |
|------|------|
| **关键词** | 出场公式、标签动作、关系定位 |
| **定义** | 角色出场 = 标签动作 + 关系定位 + 闲笔（功能性细节）。三要素完成出场即立住。 |
| **规则** | 1. **标签动作**：一个体现角色性格的动作/习惯<br>2. **关系定位**：这个角色和主角/其他人的关系<br>3. **闲笔**：一个看似随意但可能后续有用的细节 |
| **案例** | "门被踢开了。王虎大步走进来，手里的钢刀还滴着血（标签动作）。他扫了一眼堂下众人，目光在我身上停了半秒（关系定位）。他的靴子底沾满了红泥——那是城南刑场才有的土（闲笔）。" |
| **常见错误** | 1. 只有外貌描写没有动作——角色像画像<br>2. 出场了但和主角没有互动/关系——读者不知道这个人和故事有什么关系 |
| **关联卡片** | F-3, F-5, C-12 |

### 卡片 F-12：美女描写文字锚点技巧

| 字段 | 内容 |
|------|------|
| **关键词** | 美女描写、文字锚点、具象化 |
| **定义** | 描写美女时不用"倾国倾城"等空泛词，用具体可感的文字锚点制造画面感。 |
| **规则** | 1. 选取2-3个有辨识度的具体细节（不是从头到脚描述）<br>2. 细节要矛盾或有特色（美女但手上有茧/冷艳但嘴角有个小梨涡）<br>3. 用动作/神态代替形容词（"她笑的时候眼睛弯成月牙"）<br>4. 避免"沉鱼落雁""肤如凝脂"等套路化表达 |
| **案例** | 套路化："她生得倾国倾城，肤如凝脂，眉如黛。"<br>锚点化："她穿了一身素白长裙，头发只用一根木簪挽着。笑的时候左脸有个浅梨涡。右手虎口有一道疤。"——素、梨涡、疤，三个锚点比一百个形容词有用。 |
| **常见错误** | 1. 用成语/套路词描写，读者大脑中无画面<br>2. 从头描述到脚——读者记不住，没有焦点 |
| **关联卡片** | C-6, A-5, I-1 |
