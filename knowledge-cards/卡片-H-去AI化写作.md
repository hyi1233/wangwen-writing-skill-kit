# 卡片-H：去AI化写作

## 📋 知识卡片列表

### 卡片 H-1：AI四大毛病

| 字段 | 内容 |
|------|------|
| **关键词** | AI通病、四大问题、AI味识别 |
| **定义** | AI写网文最容易犯的四大通病，是"AI味"的核心来源。 |
| **规则** | 1. **滥用形容词**：璀璨、熠熠生辉、宛如…… → 解法：用最简单的普世语言<br>2. **过度解释**：动作后必加"这表现了……""仿佛……" → 解法：写完立刻停笔<br>3. **情感悬浮**：空洞的决心、抽象的悲伤 → 解法：把情感附着在具体物体上<br>4. **完美无缺**：角色无缺点无异味 → 解法：加点缺点、异味、尴尬 |
| **案例** | AI味："他的眼中闪过一丝璀璨的光芒，心中暗暗发誓一定要变得更强，仿佛一头沉睡的雄狮即将苏醒。"<br>人类味："他握紧了拳头。指甲陷进掌心。" |
| **常见错误** | 1. 四个毛病同时出现在同一段落——"AI浓度"爆表<br>2. 以为用了很多"高级词汇"就是好文笔——恰恰相反 |
| **关联卡片** | H-2, H-12, H-10 |

### 卡片 H-2：AI排毒自检清单

| 字段 | 内容 |
|------|------|
| **关键词** | 自检清单、排毒、AI味检查 |
| **定义** | 四步自检法，写完一段后逐项检查是否含有"AI味"。 |
| **规则** | 1. **有没有解释尾巴？** 动作后有没有"这表现了……""仿佛……""内心感到……"——有就删<br>2. **情感有没有附着物？** 写"他伤心"后有没有跟一个具体物象？<br>3. **人物有没有味道？** 带异味的角色才是真人（鱼腥味/汗臭/口臭）<br>4. **目的是不是太露？** 直接喊口号？改成卑微的"拜托了"或无声动作 |
| **案例** | 检查项1："他拔出刀，仿佛一头下山的猛虎。" → 删掉"仿佛"句，只留"他拔出刀。"——够了。<br>检查项3：角色是渔夫，但他身上干干净净没有鱼腥味——加一笔"一股鱼腥味从他袖口钻进鼻孔" |
| **常见错误** | 1. 自检完一条没改——"我觉得没问题"=读者觉得有AI味<br>2. 过度排毒把好描写也删了——"他笑了。"可以留着 |
| **关联卡片** | H-1, H-3, H-12 |

### 卡片 H-3：叙事自信 vs 解释强迫症

| 字段 | 内容 |
|------|------|
| **关键词** | 叙事自信、解释强迫症、留白 |
| **定义** | 人类大神相信"事情"本身就有力量（只摆画面/动作/对话），AI生怕读者看不懂必跟解释。 |
| **规则** | 1. 写一个动作/画面后——**停笔**。留给读者自己品味<br>2. 读者能自己推断出来的信息，不要替他们说<br>3. 解释是"叙事不自信"的表现——觉得自己写的东西没力量<br>4. 检验：删掉解释句后，读者还能get到原意吗？能 → 解释是多余的 |
| **案例** | AI："他把门轻轻关上，仿佛怕吵醒什么人。"<br>人类："他把门轻轻关上。"——读者知道他在怕吵醒人，不需要"仿佛"。<br>AI："她看着窗外的雨，内心感到一种说不出的孤独。"<br>人类："她看着窗外的雨。手指在玻璃上画了一个圈，又抹掉。" |
| **常见错误** | 1. 删掉了解释，但原文太简单不足以传达信息——这时不是叙事自信，是偷懒<br>2. 该解释的地方也不解释——读者确实不懂 |
| **关联卡片** | H-2, H-5, C-7 |

### 卡片 H-4：零"像"字行动铁律

| 字段 | 内容 |
|------|------|
| **关键词** | 零像字、禁用类比、铁律 |
| **定义** | 写作铁律：严禁在行文中使用"像/仿佛/宛如/好似"等情绪类比和比喻类比。 |
| **规则** | 1. 所有比喻句/类比句必须删除，换成动作/物体/反应<br>2. 不说"他像一头猛虎"——写他的行为（咆哮、猛扑、双眼充血）<br>3. 不说"她美得仿佛仙女下凡"——写具体锚点（白裙、赤足、脚踝上的红绳）<br>4. "像"字是AI味的第一识别标记 |
| **案例** | 禁用："他的眼神像刀一样锋利。"<br>改用："他盯着对方的眼睛。对方的喉结动了一下。"<br>禁用："她笑得宛如春风。"<br>改用："她笑的时候，眼角的皱纹把眼框撑成了两道弯。" |
| **常见错误** | 1. 删了"像"字但没替换成动作/反应——句子空了<br>2. 偷偷换个词（"宛若""仿佛"）——本质是一样的问题 |
| **关联卡片** | H-1, H-6, I-1 |

### 卡片 H-5：斩断解释欲

| 字段 | 内容 |
|------|------|
| **关键词** | 解释欲、动作停笔、克制 |
| **定义** | 写完一个动作或场景后，克制住补充说明/解释的冲动，立刻停笔切到下一段。 |
| **规则** | 1. 写完动作/画面后问自己：需要解释吗？读者能推断出来吗？<br>2. 能推断 → 立刻切到下一段/下一个动作<br>3. 不能推断 → 不是加解释，是前文没写好，回去修改前文<br>4. "斩断解释欲"是去AI化的核心功夫 |
| **案例** | "他把那封信扔进了火盆。"——停。不要跟"他这样做是为了彻底断绝念想"。读者知道。<br>如果他做了不寻常的事需要解释："他把那封信扔进了火盆。信还没点着，又捞出来塞进了怀里。"——通过行为跟进逻辑（C-4）解释，而不是通过旁白解释。 |
| **常见错误** | 1. 误把"行为跟进逻辑"当解释欲删掉了——行为跟进是展示逻辑，不是旁白解释<br>2. 不该斩断的地方斩断了——读者确实需要一点额外信息 |
| **关联卡片** | H-3, H-4, C-4 |

### 卡片 H-6：生理反应>心理描写

| 字段 | 内容 |
|------|------|
| **关键词** | 生理反应、心理描写替代、身体诚实 |
| **定义** | 情感先引发生理反应（干呕/胃酸/发麻/汗毛）——用生理反应替代心理描写，更真实、更有冲击力。 |
| **规则** | 1. 愤怒 → 咬肌发紧、指甲抠进掌心<br>2. 恐惧 → 胃部抽搐、后颈发凉、汗毛倒竖<br>3. 悲伤 → 喉咙发紧、眼眶发酸但不流下来<br>4. 恶心 → 胃酸上涌、干咽、嘴角不受控地抖动 |
| **案例** | 心理描写："他感到非常恐惧。"（零冲击力）<br>生理反应："他的胃突然抽搐了一下。后颈的汗毛全部竖了起来。他发现自己屏住了呼吸。"（读者跟着紧张） |
| **常见错误** | 1. 生理反应和心理描写同时出现——"他胃里一阵翻涌，心里涌起一阵恐惧"——重复了<br>2. 生理反应不准确——恐惧是胃部抽搐不是头痛 |
| **关联卡片** | H-7, F-1, I-4 |

### 卡片 H-7：情感物化锚点表

| 字段 | 内容 |
|------|------|
| **关键词** | 情感物化、锚点表、情感附着 |
| **定义** | 将抽象情感附着到具体物体上，避免情感悬浮。 |
| **规则** | 1. 思念 → 饺子（包了一碗，吃不完，放着）<br>2. 遗憾 → 旧日历（停在某一页，再翻不动了）<br>3. 父爱 → 鱼腥味（渔夫父亲身上的味道）<br>4. 初恋 → 背影（那个消失在人海中的身影）<br>5. 核心原则：情感必须绑定一个具体物/气味/声音 |
| **案例** | 不直接写"他想念母亲"。写"他端起那碗凉透的饺子，吃了两口，吃不下了。母亲包的饺子他一个人吃不完。"——思念 = 饺子。<br>不直接写"他后悔了"。写"墙上的日历还停在三月五号。从那天起就没再翻过。"——遗憾 = 旧日历。 |
| **常见错误** | 1. 情感物化不贴切——思念绑定"手机"（太现代太泛）<br>2. 物化物和情感之间的联系没有交代——读者看不懂为什么是饺子 |
| **关联卡片** | H-6, H-11, I-4 |

### 卡片 H-8："不完美才是真人"

| 字段 | 内容 |
|------|------|
| **关键词** | 不完美、缺点异味、真人感 |
| **定义** | 完美无缺的角色是AI标志之一。真人有缺点、有异味、有尴尬。 |
| **规则** | 1. 给每个重要角色至少一个缺点或异味<br>2. 异味可以是体味的（汗臭/口臭/鱼腥），也可以是行为尴尬的<br>3. 缺点不能是"太善良"这种假缺点——必须是真毛病<br>4. 主角的缺点最好是剧情相关功能性缺点（不只是一个特点） |
| **案例** | "他凑过来的时候，一股大蒜混着烟草的臭味从嘴里喷出来。"<br>"她每次撒谎都会不自觉地摸耳垂。"<br>"他打呼噜声音像拉锯，整间客栈都能听到。" |
| **常见错误** | 1. 缺点写成优点——"他太认真了""她太爱干净了"——这不是缺点<br>2. 给主角一堆缺点但不影响剧情——纯装饰 |
| **关联卡片** | H-1, F-7, F-2 |

### 卡片 H-9：杜绝上帝表达

| 字段 | 内容 |
|------|------|
| **关键词** | 上帝视角、杜绝全知、视角锁定 |
| **定义** | 杜绝用叙事者的上帝视角替角色下定义/做判断——所有评价必须由读者得出。 |
| **规则** | 1. 不写"他是一个好人"——让读者通过他的行为自己得出结论<br>2. 不写"这件事注定改变一切"——写这件事，让读者自己感受<br>3. 不在角色对话里加作者的旁白评价<br>4. 叙事者只呈现事实，不做判断 |
| **案例** | 上帝表达："这是一个残酷的世界。"<br>人类表达："城门口的饿殍排了三丈远，野狗在啃骨头。巡卫走过，连看都没看。"<br>上帝表达："他是天下最狠毒的人。"<br>人类表达："他把刀从对方喉咙里抽出来的时候，还笑了一下。" |
| **常见错误** | 1. 作者忍不住替角色总结——"她是个坚强的女人"——让读者自己看<br>2. 叙事者偶尔"泄露"一下上帝评价，破坏了整个信任体系 |
| **关联卡片** | H-3, H-10, C-7 |

### 卡片 H-10：拒绝补充解释和针对性比喻

| 字段 | 内容 |
|------|------|
| **关键词** | 拒绝解释、拒绝比喻、动作收尾 |
| **定义** | 写完一个动作立刻停笔。拒绝在该段落后面追加补充解释或针对性比喻。 |
| **规则** | 1. 写完动作 → 句号。切到下一段/下一动作<br>2. 拒绝任何形式的补充解释（"这说明他……""可以看出……"）<br>3. 拒绝任何针对性比喻（"像一头……""仿佛……"）<br>4. 如果读者没看懂——说明前文铺垫不够，回去改铺垫 |
| **案例** | 错误："他拔刀出鞘。刀身映着月光，像一条银色的蛇。他知道这一刀必须快。"<br>正确："他拔刀出鞘。刀身映着月光。"——结束。下一段写动作。 |
| **常见错误** | 1. 觉得"只写动作太干"——其实"干"就是好文笔的特征<br>2. 混淆"行为跟进逻辑"（C-4）和"解释尾巴"——前者必要，后者多余 |
| **关联卡片** | H-3, H-4, H-5 |

### 卡片 H-11：具体化原则

| 字段 | 内容 |
|------|------|
| **关键词** | 具体化、不写抽象、可感细节 |
| **定义** | 不写抽象概念，只写具体的物理细节。用"可触摸"的文字替代"可想像"的概念。 |
| **规则** | 1. 不写"他很穷"——写"他碗里的粥稀得能照出人影"<br>2. 不写"天气炎热"——写"石板路上的鸡蛋自己就能煎熟"<br>3. 不写"她很美"——写具体锚点（详见F-12）<br>4. 任何抽象形容词都可以转化为3个具体细节 |
| **案例** | 抽象："战场很惨烈。"<br>具体："断臂的士兵在泥水里爬，肠子拖了三尺远。没人扶他。" |
| **常见错误** | 1. 以为"更高级的抽象词"就是具体化——"满目疮痍"还是抽象<br>2. 细节太多冲淡了重点——一个段落3-4个锚点足够 |
| **关联卡片** | C-6, I-4, A-5 |

### 卡片 H-12：AI高频词禁区

| 字段 | 内容 |
|------|------|
| **关键词** | AI词汇、高频识别、禁用词 |
| **定义** | AI写作最爱用的那些词，一出现就能检测出"AI味"——建立一个禁用词清单。 |
| **规则** | 1. 禁用形容词：璀璨、熠熠生辉、宛如、深邃、凛冽、凛然<br>2. 禁用比喻触发：像……一样、仿佛、宛若、好似、犹如<br>3. 禁用情绪旁白：心中涌起、内心感到、眼神中透露、嘴角勾起一抹冷笑<br>4. 发现这些词 → 用动作/物体/反应替代 |
| **案例** | AI词汇检测："他心中涌起一股难以言喻的悲愤，眼神如同深渊般深邃。"（三个禁用词叠加）<br>清理后："他蹲了下去。把脸埋进双手里。指缝间有水滴下来。" |
| **常见错误** | 1. 禁用了但没找到好的替代表达——句子变得太干瘪<br>2. 只禁用了一些，偷偷留了几个同义词 |
| **关联卡片** | H-1, H-4, H-10 |

### 卡片 H-13：潜台词对话

| 字段 | 内容 |
|------|------|
| **关键词** | 潜台词、言外之意、对话层次 |
| **定义** | 对话的每句话都有表面意思和潜藏意思两个层次。好对话的精华在"没说出来"的部分。 |
| **规则** | 1. 角色说的话 ≠ 角色真正想表达的意思<br>2. 潜台词来自角色的动机、恐惧、欲望<br>3. 用动作/停顿/语气变化暗示潜台词<br>4. 好的对话让读者"听懂了角色没说的话" |
| **案例** | "你最近很忙。"——表面是观察。潜台词是"你背着我做了什么"。<br>他不答。低头把茶碗转了半圈。潜台词是"是的，被你说中了"。<br>"……那就好。"她站起来走了。潜台词是"你骗不了我"。 |
| **常见错误** | 1. 角色直接说出全部想法——没有潜台词，像在读剧本<br>2. 潜台词太隐晦——读者完全听不懂在说什么 |
| **关联卡片** | F-9, C-7, I-5 |

### 卡片 H-14：道具绑定

| 字段 | 内容 |
|------|------|
| **关键词** | 道具绑定、手不空、动作自然 |
| **定义** | 角色出场时手里必须有东西拿着/做着——"手不空"让角色动作自然、状态立体。 |
| **规则** | 1. 角色不空手：拿着杯子/擦刀/剥橘子/盘珠子/系扣子<br>2. 手里的动作反映心理状态（紧张→捏紧/放松→转动）<br>3. 道具可以成为后续伏笔（G-3）<br>4. "手不空"让人物像活人——真人手里永远有事 |
| **案例** | 坏："他坐在椅子上看着我。"<br>好："他坐在椅子上，用指甲刀一点点剪指甲。剪完一个，举在眼前看了看，才抬眼看我。"——手里的动作让角色的态度（漫不经心/不在意）一目了然。 |
| **常见错误** | 1. 角色手里拿的东西和心理状态矛盾（悠闲地嗑瓜子但说"我很紧张"）<br>2. 手里道具从头到尾没变化/没作用——纯装饰 |
| **关联卡片** | F-11, G-3, C-12 |
