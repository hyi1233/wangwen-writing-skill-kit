# 卡片-K：拆书方法论

## 📋 知识卡片列表

### 卡片 K-1：五步拆书法

| 字段 | 内容 |
|------|------|
| **关键词** | 拆书步骤、五步法、可复制 |
| **定义** | 拆书的五个步骤：选书 → 通读 → 提取重点 → 做设定集 → 倒果为因仿写。 |
| **规则** | 1. **选书**：找三江榜/L5可复制的书，不仿不可复制的神作<br>2. **通读**：约10章，记录事件团、目标、设定、人设<br>3. **提取重点**：关注数值对比/背景引导/详略技巧/行为修饰<br>4. **设定集**：人设 + 事件核心 + 金手指功能逻辑（不能跳过）<br>5. **倒果为因仿写**：先定落点 → 在前文制造起伏 |
| **案例** | 仿写步骤示例：选定一个面板流 → 通读10章记录"落差怎么造的""信息差怎么给的" → 提炼出"面板只显示熟练度但主角用它做决策"的核心逻辑 → 做出自己的设定集 → 倒果为因写好第一个事件团 |
| **常见错误** | 1. 跳过设定集直接写——写到一半逻辑崩盘<br>2. 仿写仿剧情而不是仿技法——直接照抄别人的故事线 |
| **关联卡片** | K-2, K-4, K-8 |

### 卡片 K-2：选书标准

| 字段 | 内容 |
|------|------|
| **关键词** | 选书、可复制性、扫榜 |
| **定义** | 拆书第一步——选书。核心标准是"可复制性"，不是"神作"。 |
| **规则** | 1. 选三江榜、L5写手的书——证明有人能写成，且有受众<br>2. "有一个人写成，有第二个，有十个，有一百个"——共同模式可复制<br>3. 不仿不可复制的神作：《诡秘之主》《剑来》《道诡异仙》——独一份<br>4. 扫榜看共同点，不是盯一本书觉得好就仿 |
| **案例** | 正确：选了三本都在三江榜的面板流，分析它们的共同点——都有信息差、都有落差压迫。→ 学到了共性技法。<br>错误：拆《诡秘之主》，学了一堆克苏鲁设定，结果自己完全写不出来。 |
| **常见错误** | 1. 因为自己喜欢就看某本书——不代表它能学到东西<br>2. 只看排行榜第一名——第一名往往有独特的不可复制因素 |
| **关联卡片** | K-1, K-4, K-6 |

### 卡片 K-3：设定集不能跳过

| 字段 | 内容 |
|------|------|
| **关键词** | 设定集、仿写前置、三要素 |
| **定义** | 写正文前必须完成设定集：人设 + 事件核心 + 金手指功能逻辑。跳过 = 写到崩溃。 |
| **规则** | 1. **人设**：主角和配角的性格行为模式（怎么说话、怎么决策、怎么反应）<br>2. **事件核心**：明确后续要达成的结果（落点已知才能倒果为因）<br>3. **金手指功能逻辑**：正反面分析——能干什么、不能干什么、代价是什么<br>4. 不做设定集 = "梦到哪句写哪句" = 剧情散乱 |
| **案例** | 面板流设定集：<br>人设：主角谨慎寡言，喜欢先观察再决定（影响他遇到任何事的反应）<br>事件核心：三个月后内门考核，必须达到炼气七层<br>金手指逻辑：练了就涨，但每种功法有上限；不能跨功法叠加；每次升级消耗体力 |
| **常见错误** | 1. 只写了金手指逻辑，没人设和事件核心——剧情无方向<br>2. 设定集写完不再看——设定集是写正文时的GPS，不是摆设 |
| **关联卡片** | K-1, A-1, K-5 |

### 卡片 K-4：细粮vs粗粮分类

| 字段 | 内容 |
|------|------|
| **关键词** | 细粮粗粮、签约模板、上限提升 |
| **定义** | 拆书资料按学习目的分两类：粗粮学签约模板（快速入门），细粮提升写作上限（进阶突破）。 |
| **规则** | 1. **粗粮书**：三江榜新书/中腰部签约作品 → 学"怎么写出能签约的最低标准"——快速掌握模板<br>2. **细粮书**：L5/精品/头部作品 → 学"上限在哪"——技巧/节奏/人设的天花板<br>3. 新人先用粗粮快速签约，再用细粮提升上限<br>4. 不要一开始就拆细粮书——看不懂高级技法，容易误判 |
| **案例** | 新人阶段：拆粗粮书学会"黄金三章六要素+信息差落差"——能签约。<br>进阶阶段：拆细粮书学"草蛇灰线+递进式情绪下压+多重视角"——能精品。 |
| **常见错误** | 1. 新人直接拆神作/细粮书——学了一堆花架子，基础没打好<br>2. 一直停留在粗粮阶段——签了约但写不出精品 |
| **关联卡片** | K-2, K-1, K-8 |

### 卡片 K-5：同人写作四要素

| 字段 | 内容 |
|------|------|
| **关键词** | 同人写作、四要素、节点选择 |
| **定义** | 同人写作的四个核心要素：节点选择 + 主角选择 + 风格还原 + 投其所好。 |
| **规则** | 1. **节点选择**：选原作的哪个时间/事件点切入（越早越自由、越晚越受限但读者群明确）<br>2. **主角选择**：穿越成谁？原创角色还是原著人物？<br>3. **风格还原**：模仿原作的叙事风格和语言习惯<br>4. **投其所好**：写原著读者最想看的场景/展开（弹幕最多的打法） |
| **案例** | 《斗破》同人：节点选择"退婚之前"（主角是全新角色）→ 风格还原天蚕土豆的热血升级感 → 投其所好：重点写主角和纳兰嫣然的对峙，因为这是读者最想看的 |
| **常见错误** | 1. 选了冷门节点——原著读者对这个时间点不感兴趣<br>2. 风格完全不还原——读者觉得"这不是我的同人" |
| **关联卡片** | E-10, C-15, K-7 |

### 卡片 K-6：仿写作用不仿剧情

| 字段 | 内容 |
|------|------|
| **关键词** | 仿写技法、不仿剧情、学习技法 |
| **定义** | 仿写学的是别人的写作技法（结构/节奏/信息差设计），不是照抄剧情。 |
| **规则** | 1. 仿的是"怎么写的"（技法），不是"写了什么"（剧情）<br>2. 同一个技法（如递进式情绪下压）可以用于完全不同的故事<br>3. 照抄剧情 = 抄袭，且读者会发现<br>4. 学技法是"偷师"，是合法的创作学习方式 |
| **案例** | 不仿剧情：不抄别人的面板流故事。<br>仿技法：学他怎么用信息差制造落差（先让反派嚣张→再让读者知道主角有底牌→最后反转）。技法拿走，用在完全不同的故事里。 |
| **常见错误** | 1. 仿写变成了"换皮"——换了人名但事件线一模一样<br>2. 以为仿剧情就是仿写——这是抄袭不是学习 |
| **关联卡片** | K-2, K-4, C-1 |

### 卡片 K-7："螺丝壳里做道场"

| 字段 | 内容 |
|------|------|
| **关键词** | 螺丝壳做道场、循环流、有限要素 |
| **定义** | 在有限的要素内反复循环递进——不是场景单一就是内容单调。核心是无限演绎有限要素。 |
| **规则** | 1. 选定有限要素（固定场景/固定角色群/固定事件类型）<br>2. 在要素内做无限演绎——每次循环都有新变化/新信息<br>3. 循环递进（不是循环重复）：每次比上次更深/更难/更有趣<br>4. "螺丝壳里做道场"考验的是作者的演绎功力 |
| **案例** | 重生流/循环流：主角被困在同一天。每次循环都发现新线索：第一次知道谁要杀他，第二次知道为什么，第三次知道怎么阻止——同一天的三次演绎，内容完全不同。<br>模拟推演流：同一个秘境模拟了十次，每次都换不同策略，发现不同线索。 |
| **常见错误** | 1. 循环变成了"再来一遍"——没有新信息，读者直接跳过<br>2. 循环递进太快——第三次循环就终极boss，后面没法演了 |
| **关联卡片** | C-8, J-5, B-4 |

### 卡片 K-8：认知税减免

| 字段 | 内容 |
|------|------|
| **关键词** | 认知税、普适性、备书理论 |
| **定义** | 使用读者已知的文化资产（名著/名人/历史事件）来减免理解成本——即"普适性备书"理论。 |
| **规则** | 1. 用读者已经知道的文化资产做参照，不需要重新建立认知<br>2. 同人/历史文天然享受认知税减免——读者已经知道三国/西游的人物<br>3. 原创文也可以用普适性参照（高考/职场/春运）减免认知税<br>4. 不用减免认知税 = 每个概念都要解释 = 前三章全是明 |
| **案例** | 三国同人：读者已经知道曹操/刘备/关羽——不需要介绍。可以直接写"主角穿越到赤壁之战后给刘备出主意"。<br>原创修仙文："这个考试就像前世的高考一样。"——用高考减免"修仙考核"的认知税。 |
| **常见错误** | 1. 完全自创体系，每个概念都要解释——认知税太高<br>2. 用冷门文化资产——读者不知道，认知税没有减免 |
| **关联卡片** | C-3, A-4, K-5 |
