# 卡片-L：简介写作

## 📋 知识卡片列表

### 卡片 L-1：三段式标准架构

| 字段 | 内容 |
|------|------|
| **关键词** | 简介架构、三步法、标准模板 |
| **定义** | 简介 = 开局钩子（困境/反常） → 核心看点（金手指出现） → 全书脉络（主角立场/核心梗）。 |
| **规则** | 1. **第一步：开局钩子**——开篇事件简述，含困境或脑洞，几十到上百字，颠覆预期<br>2. **第二步：核心看点**——金手指出现，让读者感知两者关联能解决事件<br>3. **第三步：全书脉络**——主角立场/核心梗，读者寻找认同的剧情倾向<br>4. 三步骤是最经典的简介写法，适合大多数题材 |
| **案例** | 钩子："穿越第一天，全家被抄了。"（困境+反常）<br>金手指："还好我有一个面板，只要练就能涨修为。"（解决途径）<br>全书脉络："既然如此，我就用这面板，从这烂牌打出一番天地。"（主角立场） |
| **常见错误** | 1. 只有钩子没有金手指和全书脉络——读者不知道看点在哪<br>2. 三要素的顺序僵化——可以根据题材调整顺序，但要素不能少 |
| **关联卡片** | L-2, L-3, A-11 |

### 卡片 L-2：三要素不可少

| 字段 | 内容 |
|------|------|
| **关键词** | 三要素、简介底线、不可缺失 |
| **定义** | 简介可以变顺序、可以自己演绎，但困境/金手指/全书脉络三要素缺一不可。 |
| **规则** | 1. 三要素 = 困境概述 + 金手指出现 + 全文脉络/最终目标<br>2. 顺序可变：也可以先金手指后困境（"我有面板，但我全家被抄了"）<br>3. 演绎可变：可以用对话/场景/独白来承载三要素<br>4. "文无定式，要素不能缺失"——没有要素的简介是无效简介 |
| **案例** | 正常顺序：困境 → 金手指 → 脉络<br>变序："我有一个能无限升级的面板。可惜，我穿越到了全族被灭的那一刻。不过，有面板，一切好说。"（金手指→困境→脉络） |
| **常见错误** | 1. 用了一堆文艺的话写简介，三要素一个都没覆盖<br>2. 以为"悬念"可以替代三要素——没有金手指的简介对网文无效 |
| **关联卡片** | L-1, L-3, A-11 |

### 卡片 L-3：简介两种写法

| 字段 | 内容 |
|------|------|
| **关键词** | 简介写法、三步架构、噱头反常 |
| **定义** | 简介有两种核心写法：三段式架构（稳妥） vs 噱头反常式（冲击力）。 |
| **规则** | 1. **三步架构**：困境→金手指→脉络（详见L-1），适合大多数题材<br>2. **噱头反常式**：用一个反常/颠覆预期的事件做钩子，再连接金手指和脉络<br>3. 选择标准：传统升级流/系统流用三步架构；脑洞流/反套路用噱头反常<br>4. 两种写法三要素都齐，只是钩子的呈现方式不同 |
| **案例** | 三步架构：见L-1<br>噱头反常："别人穿越带系统。我穿越带了一本《五年高考三年模拟》。"（反常钩子）→ 后续连接"在异世界当教书先生"的脉络 |
| **常见错误** | 1. 噱头反常但三要素不全——有噱头没有金手指<br>2. 选了噱头反常但噱头和正文风格不一致——读者觉得被骗 |
| **关联卡片** | L-1, L-2, A-12 |

### 卡片 L-4：不同类型简介要点

| 字段 | 内容 |
|------|------|
| **关键词** | 简介分类、不同类型、要点差异 |
| **定义** | 不同类型小说的简介有不同的侧重点和写法要点。 |
| **规则** | 1. **背景设定类**：简介里的背景必须是主角将要行进的主要事件，与剧情无关的全是废话<br>2. **同人/历史文**：抓住开篇节点+全书脉络即可——读者已经知道世界观<br>3. **系统流/面板流**：突出金手指和金手指能解决什么具体问题<br>4. **非新人模板**：要根据题材调整，不能一个模板通吃 |
| **案例** | 历史文简介不需要解释历史背景——读者知道三国。"穿越成刘禅，诸葛北伐前夜。我有一个面板，能看到所有人的忠诚度。"够了。<br>原创修仙文简介：必须先建立世界观。"这个世界，修炼=烧寿元。多练一次功，少活十年。但我的金手指是——练功加寿元。" |
| **常见错误** | 1. 用一个简介模板套所有题材——不合脚<br>2. 同人简介写了一半世界观介绍——读者不需要 |
| **关联卡片** | L-1, L-2, K-5 |

### 卡片 L-5：风格一致性是沉浸感基石

| 字段 | 内容 |
|------|------|
| **关键词** | 风格一致、沉浸感、书名简介正文统一 |
| **定义** | 书名→简介→正文三层的风格必须一致，否则读者产生认知断裂，沉浸感破灭。 |
| **规则** | 1. 书名的风格基调 → 简介必须延续 → 正文必须确认<br>2. 书名搞笑风格，简介严肃风格 → 读者困惑<br>3. 简介文艺风格，正文流水账 → 读者觉得被骗<br>4. 一致性 = 沉浸感的基石，断裂 = 读者出戏 |
| **案例** | 一致：书名《我的面板能无限升级》→ 简介"面板在手天下我有"→ 正文面板流叙事。风格统一，读者进入状态。<br>不一致：书名《我的面板能无限升级》→ 简介写了一大段文艺的古风独白 → 读者不知道是仙侠还是面板流。 |
| **常见错误** | 1. 简介为了吸引眼球用了和正文完全不同的风格<br>2. 作者觉得"简介要文艺一点才有格调"——网文不需要格调需要匹配 |
| **关联卡片** | A-11, A-12, L-1 |

### 卡片 L-6：反差式身份揭示

| 字段 | 内容 |
|------|------|
| **关键词** | 反差揭示、身份反转、简介钩子 |
| **定义** | 简介中通过反差式的身份揭示制造钩子——先建立一个身份预期，然后揭示完全相反。 |
| **规则** | 1. 先给出一个身份标签或场景暗示<br>2. 紧跟一个反差揭示，颠覆预期<br>3. 反差必须连接到金手指或全书脉络<br>4. 这是简介噱头反常法（L-3）的一个具体应用变体 |
| **案例** | "我是这个宗门里最不起眼的扫地弟子。直到有一天，长老们发现我扫的每一把扫帚，都成了上古神器。"<br>（身份：扫地弟子 → 反差：扫帚成神器 → 暗示金手指：他碰过的东西会升级？） |
| **常见错误** | 1. 反差揭示很搞笑但和正文无关——纯标题党<br>2. 反差太离谱——读者觉得"这不可能"，不点进来看 |
| **关联卡片** | L-3, A-12, B-12, E-8 |
