# 卡片-I：文字质量

## 📋 知识卡片列表

### 卡片 I-1：删"是"字加动作

| 字段 | 内容 |
|------|------|
| **关键词** | 删是字、动词驱动、去静态 |
| **定义** | 把行文中所有"是/是……的"结构删掉，替换为动作描写，让文字从静态变为动态。 |
| **规则** | 1. 遇到"他是很生气的" → 改为动作"他把桌子一拍，茶水四溅"<br>2. 遇到"这是一个危险的地方" → 改为"地上的血迹还没干"<br>3. "是"字句 = 定义句 = 告诉读者 = AI味<br>4. 替换时不需要每个"是"都改（判断句/定义句保留），改描写性"是" |
| **案例** | "他是愤怒的。他是一个很强大的对手。"<br>→ "他咬紧了后槽牙。那人站在阴影里，只露出半张脸，但光是那股气场就让人喘不过气。" |
| **常见错误** | 1. 所有"是"都删，连该保留的判断句也删了——句子不通<br>2. 换了动词但不准确——"他把桌子一踢"（应该"拍"不是"踢"） |
| **关联卡片** | H-4, H-11, E-5 |

### 卡片 I-2：动词精准有力

| 字段 | 内容 |
|------|------|
| **关键词** | 动词精准、去副词、力量感 |
| **定义** | 用精准有力的动词替代"副词+弱动词"组合，让每个动作有力量感和画面感。 |
| **规则** | 1. "他快速地跑了" → "他冲了出去"（副词+弱动词 → 精准动词）<br>2. "他紧紧握住刀" → "他攥住刀柄"（攥=紧+握）<br>3. "他慢慢走过" → "他踱过"（踱=慢+走）<br>4. 动词越精确，需要的修饰词越少 |
| **案例** | 弱动词："他用力地踢开了门。"<br>强动词："他踹开了门。"<br>弱动词："他快速地看了一眼周围。"<br>强动词："他扫了一眼。" |
| **常见错误** | 1. 选了不准确的大词动词——"他翱翔在战场上"（翱翔=飞行，用在人身上不合适）<br>2. 过度追求生僻动词，反而不自然 |
| **关联卡片** | I-1, H-11, F-1 |

### 卡片 I-3：生活化比喻

| 字段 | 内容 |
|------|------|
| **关键词** | 生活化、接地气、比喻降维 |
| **定义** | 用生活化的事物做比喻，不用文学化比喻。网文比喻的核心原则是"读者一看就懂"。 |
| **规则** | 1. 比喻要来自普通读者的生活经验（吃饭/赶集/打工/高考）<br>2. 不用文学化典故（不引经据典、不说"犹如夸父追日"）<br>3. 穿越者可以做"现代 vs 异世界"的比喻，天然有趣<br>4. 比喻的目的是降低理解门槛，不是展示文采 |
| **案例** | 文学化："他的剑法如行云流水，宛若公孙大娘舞剑。"<br>生活化："他的剑法跟切豆腐似的，一刀下去连渣都不剩。"<br>"修炼就像高考，背得多不一定考得好，但背得少肯定考不好。" |
| **常见错误** | 1. 比喻来自冷门典故，读者get不到<br>2. 比喻太复杂，理解比喻比理解原概念还费劲 |
| **关联卡片** | C-3, F-8, H-11 |

### 卡片 I-4：多感官细节

| 字段 | 内容 |
|------|------|
| **关键词** | 多感官、视觉听觉嗅觉触觉、立体描写 |
| **定义** | 不只写视觉，调动听觉/嗅觉/触觉/味觉等多种感官，让场景立体化。 |
| **规则** | 1. 视觉：最基本，每段都有<br>2. 听觉：声音/动静/沉默——增加氛围<br>3. 嗅觉：气味是最强记忆触发器（血腥味/花香/霉味）<br>4. 触觉：温度/质感（粗糙/冰冷/粘腻）——最被忽视的感官<br>5. 每段至少含两种以上感官 |
| **案例** | 纯视觉："他走进了一间阴暗的地下室。"<br>多感官："他走进地下室。一股霉味呛得他鼻子发酸（嗅觉）。脚下的石板又湿又滑（触觉）。远处有水滴声，嗒、嗒、嗒（听觉）。" |
| **常见错误** | 1. 全篇只有视觉——像在看无声电影<br>2. 五感一次全塞进一段——信息过载。控制在2-3个/段 |
| **关联卡片** | H-6, H-11, I-9 |

### 卡片 I-5：三层语言系统

| 字段 | 内容 |
|------|------|
| **关键词** | 三层语言、叙述层对话层内心层、语言分层 |
| **定义** | 叙述语言（客观）+对话语言（角色个性）+内心独白语言（主观私人）——三个层次风格不同。 |
| **规则** | 1. **叙述层**：简洁客观，信息密度高，不带情绪<br>2. **对话层**：因角色身份而异（F-9语言DNA），口语化<br>3. **内心独白层**：主观化，可以用短句/碎片句/感叹，带情绪<br>4. 三层不能混——不能用叙述层的风格写对话 |
| **案例** | 叙述层："三天后。李家庄。"（干净利落）<br>对话层："你这狗日的，欠我的钱什么时候还？"（口语化、带情绪）<br>内心独白层："得走。现在就。再不走来不及了。"（碎片、紧张） |
| **常见错误** | 1. 三层语言风格一样——对话像旁白，旁白像对话<br>2. 内心独白写得太完整太理性——人在紧张时思维是碎片化的 |
| **关联卡片** | F-9, H-13, C-15 |

### 卡片 I-6：短-短-长-短节奏交替

| 字段 | 内容 |
|------|------|
| **关键词** | 短句节奏、长短交替、句式韵律 |
| **定义** | 通过短句和长句的交替排列，创造文字节奏感。通用模式：短→短→长→短。 |
| **规则** | 1. 短句 = 鼓点/冲击（紧张/强调）<br>2. 长句 = 流水/铺垫（缓和/解释）<br>3. 交替使用产生韵律感，读者读起来有节奏<br>4. 高潮时多用短句（加速），平静时多用长句（放缓） |
| **案例** | "风起。门关了。屋里安静得能听见自己的心跳。他在黑暗里等了很久。"（短-短-长-短）<br>紧张场景多用短句连击："刀来。血溅。人倒。没人动。" |
| **常见错误** | 1. 全是长句——读者读着读着就困了<br>2. 全是短句——读起来太碎，没有连贯性 |
| **关联卡片** | J-1, I-7, I-8 |

### 卡片 I-7：手术刀式分段

| 字段 | 内容 |
|------|------|
| **关键词** | 分段技巧、短句成行、节奏分割 |
| **定义** | 用分段（换行）代替标点来创造节奏。一个动作/细节独立成段 = 强调。 |
| **规则** | 1. 重要的动作/细节独立成段，产生视觉冲击<br>2. 连续短句连续分段——像电影的快切镜头<br>3. 段落越短，阅读速度越快<br>4. 网文读者用手机阅读，短段落更易读 |
| **案例** | "他推开了门。<br><br>屋里没人。<br><br>桌上放着一封信。信封口开着。<br><br>他走过去，拿起信。"——每段一个动作，节奏明快，读者一页一页翻得快。 |
| **常见错误** | 1. 所有段落都是五六行长段——手机阅读体验极差<br>2. 分段太碎（每句都分段）——像诗不像网文 |
| **关联卡片** | I-6, I-9, J-3 |

### 卡片 I-8：拟声词占行充当鼓点

| 字段 | 内容 |
|------|------|
| **关键词** | 拟声词、鼓点效果、独立成行 |
| **定义** | 用拟声词独立占一行，充当文字节奏中的"鼓点"——强调、加速、营造氛围。 |
| **规则** | 1. 拟声词单独成行：咚/啪/嗒/嗡<br>2. 用在关键动作的瞬间——刀砍中/门关上/雷响起<br>3. 多个拟声词连续占行可制造连击感<br>4. 用多了失去效果——一场戏最多3-4个 |
| **案例** | "他举起了刀。<br><br>嗡——<br><br>刀锋劈下。<br><br>嗒。<br><br>一滴血落在地上。" |
| **常见错误** | 1. 拟声词用在无关紧要的地方——削弱了冲击力<br>2. 滥用——每段都有拟声词，读者麻木了 |
| **关联卡片** | I-6, I-7, I-9 |

### 卡片 I-9：恐怖/反转最后一句必须短句定格

| 字段 | 内容 |
|------|------|
| **关键词** | 短句定格、恐怖收尾、最后一句 |
| **定义** | 恐怖场景和反转场景的最后一句话必须是短句——短到像钉子一样钉在读者脑中。 |
| **规则** | 1. 恐怖场景最后一句 ≤ 10个字<br>2. 反转场景最后一句 ≤ 12个字<br>3. 短句定格 = 画面定格 = 让读者在翻页前脑子里反复回响<br>4. 最后一句不能有新信息——它只是把已有的信息收束 |
| **案例** | 恐怖："他掀开被子。床上没有人。但枕头是热的。"（短句收束，毛骨悚然）<br>反转："他等了十年的那个人，是他自己。"（短句收束，恍然大悟） |
| **常见错误** | 1. 最后一句太长——"他突然发现，原来床上的人就是他十年来一直寻找的那个人的弟弟。"——冲击力全没了<br>2. 最后一句加了新信息——读者还在消化，没来得及回味就翻页了 |
| **关联卡片** | E-8, I-6, I-8 |

### 卡片 I-10：方言作为调味剂

| 字段 | 内容 |
|------|------|
| **关键词** | 方言、调味剂、适度使用 |
| **定义** | 适度在对话中加入方言词汇，增加地域真实感和角色辨识度——但只是调味剂，不能当主食。 |
| **规则** | 1. 每段对话最多1-2个方言词，多了读者看不懂<br>2. 方言词必须是"可猜"的——即使不认识也能从上下文猜意思<br>3. 用方言强化角色身份（粗汉/老农/江湖人）<br>4. 叙述层不用方言——只用在对话层（I-5三层语言系统） |
| **案例** | "龟儿子，你敢来？"（四川）——读者知道是骂人话。一个"龟儿子"点明地域，其余用普通话保证理解。<br>"伢子，来吃碗嘞。"（湖南）——"伢子"=小孩，"嘞"=这里，可猜。 |
| **常见错误** | 1. 整段对话全是方言——除了当地人没人看懂<br>2. 叙述层也用方言——读起来太累 |
| **关联卡片** | F-9, I-5, C-3 |

### 卡片 I-11：恐怖物理递进

| 字段 | 内容 |
|------|------|
| **关键词** | 恐怖递进、听觉震动视觉、层级推进 |
| **定义** | 恐怖场景按物理感知的层级递进：听觉 → 震动/触觉 → 视觉，每一步加深恐惧感。 |
| **规则** | 1. 第一层（听觉）：先听到什么（脚步声/滴水声/呼吸声）<br>2. 第二层（震动/触觉）：感觉到什么（地面微震/冷气/发麻）<br>3. 第三层（视觉）：最后看到什么（真相/怪物）<br>4. 这个顺序模拟了人在黑暗中的感知过程——真实感倍增 |
| **案例** | "嗒。嗒。嗒。"（听觉：脚步声）<br>"脚下的石板微微颤动。"（震动：有什么重的东西在靠近）<br>"他举起油灯。光扫过一个佝偻的身影，拖了三条影子。"（视觉：真相） |
| **常见错误** | 1. 直接跳到视觉——"他看到了一只怪物"——没铺垫，不恐怖<br>2. 递进太多层——超过三层读者就疲劳了 |
| **关联卡片** | I-4, I-9, A-5 |

### 卡片 I-12：恐怖质感必须有触觉

| 字段 | 内容 |
|------|------|
| **关键词** | 恐怖触觉、粘腻冰冷、恐怖质感 |
| **定义** | 恐怖场景必须包含触觉描写（粘腻/冰冷/发麻/刺痛）——触觉是恐惧感最深的触发器。 |
| **规则** | 1. 视觉告诉读者"有什么"，触觉告诉读者"有多恐怖"<br>2. 经典触觉词：冰冷/粘腻/粗糙/湿滑/刺痛/发麻/温热<br>3. 触觉放在恐怖场景的中段（I-11物理递进的第二层）<br>4. 触觉比视觉更早触发人的恐惧本能 |
| **案例** | 仅有视觉："墙角站着一个白衣女子。"<br>加上触觉："墙角站着一个白衣女子。他走过去的时候，一阵冷风从她脚底钻过来，顺着裤管爬上了小腿。"——冷风的触感让恐惧从画面变成了体验。 |
| **常见错误** | 1. 恐怖场景只有视觉——像在看照片，不是体验<br>2. 触觉描写太生硬——"他觉得很害怕"（这是情绪不是触觉） |
| **关联卡片** | I-4, I-11, H-6 |
